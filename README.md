# NexusERP

A comprehensive ERP (Enterprise Resource Planning) system featuring a **Student Progress Management System (SPMS)** module for tracking and managing student academic progress.

## Features

- Student progress tracking and reporting
- Data management and analytics
- Web interface for easy access

## Tech Stack

- **Python** — core backend logic
- **HTML / CSS / JavaScript** — frontend interface
- **C / Cython** — performance-critical components

## Getting Started

Clone the repository:
git clone https://github.com/AvadhutVG/NexusERP.git
cd NexusERP
